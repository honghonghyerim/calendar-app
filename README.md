# 일정관리 프로그램 (Java)

Java 기초부터 객체지향(OOP), 단위 테스트까지 학습하며 제작한 콘솔 기반 일정관리 프로그램입니다.  
사용자가 일정을 추가, 조회, 수정, 삭제할 수 있습니다.

---

## 프로젝트 소개
- Java 기초와 객체지향 개념 실습용 프로젝트
- 일정 데이터를 날짜별로 관리
- CRUD 기능 제공 (생성, 조회, 수정, 삭제)


---

## 주요 기능
- 일정 추가(Add Schedule)
- 일정 조회(View Schedule by Date)
- 일정 수정(Update Schedule)
- 일정 삭제(Delete Schedule)
- 날짜별 일정 관리
